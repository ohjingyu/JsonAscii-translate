# JsonAscii-translate

개발자 변환 도구 — 한 페이지에서 다 처리

## 기능
- **JSON**: 정렬 / 압축 / 문자열 인코딩·디코딩
- **ASCII**: 텍스트 ↔ 코드 변환, 0–127 검색 가능한 표
- **진수 변환**: 2 / 8 / 10 / 16진수 실시간 변환 (BigInt 지원)
- **URL**: encodeURI / encodeURIComponent / decodeURIComponent + URL 파서
- **QR 코드**: 생성(PNG 다운로드) + 이미지에서 디코딩 (드래그·붙여넣기 지원)
- **Unicode**: UTF-16 / ES6 / HTML / UTF-8 형식 변환, 문자별 상세표
- **Base64**: 텍스트/파일 인코딩·디코딩, URL-safe, Data URL

## 사용
`index.html` 파일 하나로 동작합니다. 브라우저에서 바로 열거나, 정적 호스팅(Cloudflare Pages 등)에 올리세요.
